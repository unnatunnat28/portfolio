# 🌟 Unnat Singh - Portfolio Website (MERN Stack)

A modern, responsive **portfolio website** built with the **MERN stack**, featuring a contact form with email notifications (via Resend API), dark/light theme toggle, and sections for projects, skills, certifications, and mission-critical work experience.

🔗 **Live Frontend:** [ayodhyaestate.com](https://ayodhyaestate.com/)  
🔗 **Live Backend:** [egamanportfolio-backend.onrender.com](https://egamanportfolio-backend.onrender.com)

---

## 🚀 Features

- 💻 Responsive Design (mobile-first)  
- 🌙 Dark/Light Theme toggle  
- ✉️ Contact Form with instant email notifications via Resend API  
- 🏆 Project Showcase with links  
- 🛠️ Skills Section  
- 🎓 Certifications display  
- 👨‍💻 Coding Platform links (LeetCode, HackerRank, GFG)  
- ✨ Smooth animations & scroll effects  

---

## 🛠️ Tech Stack

**Frontend:** React.js, Axios, CSS3, Font Awesome, Context API  
**Backend:** Node.js, Express.js, Resend API, CORS
